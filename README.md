# 🛍️ Thrift By Yourself (WIP)

A backend application built using Node.js and Express.js that allows users to manage and list fashion products. The application uses MongoDB for storing product data and provides RESTful CRUD API endpoints.

> 🚧 This project is currently a Work in Progress (WIP)

---

## 🚀 Features

- Create product listings (name, price, quantity, description)
- Fetch all products
- Update product details
- Delete products

---

## 🛠️ Tech Stack

- Node.js
- Express.js
- MongoDB
- Mongoose

---

## 📡 API Endpoints

- `POST /api/product` → Add a product  
- `GET /api/product` → Get all products  
- `PUT /api/product/:id` → Update product  
- `DELETE /api/product/:id` → Delete product
